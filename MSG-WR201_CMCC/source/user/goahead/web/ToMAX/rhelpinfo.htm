端口VLAN

VLAN（虚拟局域网）可以将网络逻辑地分割成多个不同的广播域，从而有助于控制流量、简化网络管理、加强网络安全性

端口VLAN是将不同的端口划分到一个VLAN组内，属于同一个VLAN的端口之间可以相互通信，不同VLAN的端口之间不能通信
注意 一个端口可以属于多个VLAN，默认情况下所有端口都属于一个VLAN，VLAN组号为1

端口汇聚

把2个或多个交换端口聚合在一起，形成一个高带宽的数据传输通道。聚合在一起的所有端口看作一个逻辑端口，工作起来像一条通道一样，该逻辑端口带宽为汇聚组内所有端口带宽的叠加
注意 1、同一汇聚组内的端口的端口参数必须一致，端口速率、双工模式必须相同，且必须完全属于某一VLAN
2、若设备设置了TAG VLAN，则同一汇聚组内的端口必须具有相同的TAG VLAN端口设置
3、汇聚组内不能含有监控端口
4、同一汇聚组内的端口的速率限制必须相同
5、已开启LACP或RSTP的端口，无法设置汇聚

端口镜像

将交换机其他端口的流量自动复制到监控端口，实时提供各端口的传输状况的详细资料，以便网络管理人员进行流量监控、性能分析和故障诊断
监控端口 对被监控端口流量进行监控的端口，监控端口只能有一个
被监控端口 可选择一个或多个端口
注意 1、若开启端口镜像功能则被监控端口不能与监控端口是同一个端口
2、如果某端口已经属于某个汇聚组，则此端口不能再设置为监控端口（监控端口下拉框中的红色字体的端口表示该端口属于汇聚组中）
